# UnblockMe – Website

Static one-page site for UnblockMe (Auckland). Deployed to Netlify.

## Local edit
- Edit `index.html`.
- Forms: posts to FormSubmit → sends to `unblockmeltd@outlook.co.nz`.

## Deploy on Netlify
1. Create a new GitHub repo and push this folder.
2. Netlify → Add new site → Import from Git → pick repo.
3. Build command: *(leave blank)*
4. Publish directory: `.`
5. Add custom domain and enable HTTPS.

## Email
FormSubmit will email to unblockmeltd@outlook.co.nz.
